# Banking System Master (MERN Stack)

## Overview

Banking System Master is a full-stack web application developed using the MERN stack (MongoDB, Express.js, React, and Node.js). The application simulates core banking functionalities such as account management, money transfers, transaction tracking, and user authentication.

## Features

- User Registration and Login
- Secure Authentication using JWT
- Customer Account Management
- Deposit and Withdrawal Operations
- Fund Transfer Between Accounts
- Transaction History Tracking
- Responsive Dashboard
- RESTful API Architecture
- MongoDB Database Integration

## Tech Stack

### Frontend
